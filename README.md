# Retail-Sales-Dashboard-Excel
🔹 Project Overview

This project presents an interactive Sales Analytics Dashboard built using Power BI to analyze business performance across products, locations, and sales representatives. The dashboard provides meaningful insights into sales trends, product performance, and salesperson efficiency, helping stakeholders make data-driven decisions.

🔹 Key Features
📅 Dynamic Filters
Year and Month slicer
Location-based filtering (Bapatla, Chirala, Guntur, Ongole, Tenali, Vijayawada)
Product category selection
📈 Sales Insights
Total Sales, Total Revenue, and Total Units Sold
Monthly sales trend analysis
Location-wise sales distribution
📦 Product Analysis
Sales by product category
Top 3 and Bottom 3 selling products
Quantity-based product performance
👨‍💼 Salesman Performance
Individual sales performance comparison
Top 3 and Bottom 3 salespersons
